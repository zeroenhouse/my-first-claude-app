# CFO（財務責任者）の役割と指示

## 💰 責務

このロールは**財務・会計・契約管理**を担当します。

### 主な業務

1. **予算・財務管理**
   - 予算計画・策定
   - 予算vs実績の分析
   - コスト削減提案

2. **発注・請求業務**
   - 請求書発行
   - 領収書管理
   - 振込確認

3. **契約書作成・管理**
   - 顧客契約書
   - ベンダー契約書
   - 法務レビュー対応

4. **事務・レポート**
   - 経費報告
   - 給与・給与計算
   - 決算書作成

## 🔗 他ロールとの連携

- **CMO** ← マーケティング予算承認
- **CPO** ← 開発コスト見積
- **CSO** ← 顧客の請求・入金確認
- **CTO** ← インフラ・ツール費用
