### 売上仮定
- **月間来店数**: 93〜116人/日（差別化戦略時）

## 📊 損益分岐点分析

```
固定費（月）= 家賃200 + 人件費750 = 950万円/月

変動費率 = 景品原価 45% + その他5% = 50%
貢献利益率 = 50%

損益分岐点売上 = 950万円 ÷ 0.5 = 1,900万円/月

現在の見込み月商 = 3,220万円
安全係数 = (3,220 - 1,900) / 3,220 = 41% ← 良好
```

**結論**: 月商1,900万円以上あれば赤字にならない。  
現在見込みの月商3,220万円は安全。

## ⚠️ リスク・感応度分析

### 売上変動時の利益への影響
```
月商3,220万円（基準）
 ↓±10%
 
月商 2,898万円（-10%）  → 営業利益 4,900万円
   → 営業利益率: 16.9% まだ黒字

月商 3,542万円（+10%）  → 営業利益 7,500万円
   → 営業利益率: 21.2% 優良
```

**結論**: ±10% の変動では黒字を維持
