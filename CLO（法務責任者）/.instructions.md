GiGO我孫子クレーンゲーム専門店への転換にあたり、以下の法的リスクを管理する：

1. **買収・M&A 契約**
2. **不動産賃貸借契約**

7. **税務申告・会計**

## ⚠️ 重要な法的リスク Map

### Risk 1: 景品表示法・風営法対応 🔴 [HIGHEST PRIORITY]

#### 1-1. 景品表示法（景表法）
```
【規制内容】
クレーンゲーム = 景品取得ゲーム として景表法規制の対象

【重要ポイント】
├ 景品の種類・価値に上限規制あり
├ 取得確率の表示が要件
├ 「取れやすさ」の実装に法的基準あり
└ 違反時のペナルティ: 業務停止命令・罰金

【対応方法】
□ 消費者庁の「景品表示法ガイドライン」を精読
□ 景品の種類・提供条件を確認申請
□ 爪の強度・取得難度を適法範囲内に設定

【チェックリスト】
□ 各景品の表示価格 ≤ 法定上限額（カテゴリ別）
□ 「取得確率X%」などの表示が正確か
□ 提供条件を消費者に明示
□ 年1回の景品監査（専門家による）
```

#### 1-2. 風営法との関係
```
【日本の法律】
クレーンゲーム = 「ゲームセンター」として風営法の枠外
※ パチンコやスロットとは異なり「遊技」ではなく「娯楽」扱い

【CLO確認事項】
√ 千葉県：クレーンゲーム専門店の営業許可は不要
※ 地域によって異なるため要確認

【アクション】
□ 千葉県警察本部 > 風営課に事前相談
□ 我孫子警察署に営業予定を事前届出（慣例）
□ 景品内容を届出
```

### Risk 2: 賃貸借関連 🔴 [HIGH PRIORITY]

#### 2-1. 現在の負担者確認（重要！）
```
【現況】
├ GiGO我孫子は「スマイルフィールド我孫子」内に入居
├ VZONE（パチンコ）と同一施設内
└ 施設管理者 = スマイルフィールド管理会社

【リスク】
GiGO（現賃借人）からの転貸でなく、
直接スマイルフィールドとの新規契約が必要

【必要書類】
□ 現在の賃貸契約書（GENDA ← スマイルフィールド）
□ スマイルフィールド管理会社の同意書
□ 転貸不可条項の確認
```

#### 2-2. 新規賃貸借契約の条件交渉
```
【交渉項目】
├ 賃料：現在のGiGO賃料をベースに交渉
├ 契約期間：5年 or 10年？
├ 敷金：月賃料の3〜6ヶ月分
├ 保証金：追加で確認
├ 改装費の負担：誰が負担するか
├ 原状回復義務：どの程度か
└ 中途解約条項：あるか、違約金は？

【CLOチェック】
□ 契約期間終了時の買い直し可能性
□ 賃料の改定条項（2年ごと？）
□ 不可抗力条項（天災・パンデミック時）
```

#### 2-3. 電気・水道・ガス
```
【確認事項】
├ VZONEとの共用設備の負担分
├ 電気容量（クレーン460台で冷房稼働）
├ 給排水（景品清掃・トイレ）
└ 廃棄物処理（景品梱包・開封後の空き箱）

【契約】
□ 各ユーティリティの使用契約（名義確認）
□ スマイルフィールドとの配分合意書
```

### Risk 3: 買収契約（M&A） 🔴 [HIGH PRIORITY]

#### 3-1. 買収対象確認
```
現在の仮定: 「営業権の買収 5,000万円」

【具体的には何を買収するのか？】
□ GENDA GiGO Entertainment の「営業権」か
□ 在庫（景品・ストック）か
□ 固定資産（機器）か
□ 顧客データベースか
□ 従業員（引き継ぎ）か

【リスク】
├ 隠れた債務（未払い金・ローン残額）
├ 環境汚染・建物瑕疵
└ 契約関連の引き継ぎ義務
```

#### 3-2. 買収契約書の必須条項
```
【基本項目】
□ 買収日の確定（決済日）
□ Purchase Price: 5,000万円（分割払いの有無）
□ クロージング条件（融資承認など）
□ 売主の表明・保証（hidden liabilities ないか）
□ 買主の表明・保証
□ 調査（デューデリジェンス / DD）期間

【特に注意】
├ フランチャイズ料の残債がないか
├ 機器メーカーとの契約残存期間
├ スタッフ雇用契約の引き継ぎ
└ 顧客データの取扱い（個人情報）
```

#### 3-3. DD（デューデリジェンス）項目
```
【財務DD】
□ 過去3年の月商推移・決算書
□ 未払い金・再検査あるか
□ 給与未払いないか
□ 上納金（加盟店料）の支払い状況

【法務DD】
□ 賃貸借契約書確認
□ 各種許認可・届出の状況
□ 訴訟・行政処分 ないか
□ 契約関連（機器リース等）の引き継ぎ

【商業DD】
□ 顧客層・リピート率
□ 機器の状況（古さ・メンテ状況）
□ ROI / 回収期間の実績
```

### Risk 4: 労務管理・採用 🟡 [MEDIUM PRIORITY]

#### 4-1. 既存スタッフ引き継ぎ
```
【重要】
現GiGO我孫子に「スタッフ1名」と記載

【リスク】
├ その1名を引き継ぐか or 新規採用か
├ 引き継ぐ場合の労働条件（給与・身分）変更
├ 退職金・転籍手当の必要性
└ 勤続年数・厚生年金の扱い

【CLO対応】
□ 既職員 1名との雇用契約書確認
□ 継続雇用を希望するか確認
□ 新契約書の作成（身分変更など）
```

#### 4-2. 新規採用規制
```
【必須確認】
├ 募集・採用時の差別禁止
│  └ 年齢制限（最高年齢 or 「若者向け」等の表現）
│  └ 性別制限（女性/男性のみなど）
│
├ 最低賃金の遵守
│  └ 千葉県最低賃金（2025年：1,150円予定）
│  └ 給与計算に含まれない費用（食事代など）
│
├ 労働保険への加入
│  └ 雇用保険・労災保険の加入手続き
│
├ 給与明細の適正発行
│  └ 控除項目の合法性確認
│
└ 残業管理・36協定
   └ 残業時間の上限規制（月45時間 etc）
```

#### 4-3. シフト管理
```
【計画内容】
現在: スタッフ29名、月間人件費750万円

【リスク】
├ 月750万円は本当に実現可能か？
├ 所定労働時間（月○時間）で何時間の給与か
├ 残業規制に準拠しているか
└ 深夜勤務手当の支払い漏れ

【CLO対応】
□ 労働時間の総額制約を確認
□ 繁忙期の残業シミュレーション
□ 就業規則の作成（採用前に）
```

### Risk 5: 消費者保護・クレーム 🟡 [MEDIUM PRIORITY]

#### 5-1. 景品の不良品・交換対応
```
【消費者契約法との関連】
顧客が「取得した景品に不具合」→ ショップ責任？

【対応方針】
├ 景品原価で提供 = 「新品未検査」を明示
├ 「取得後の検査は顧客責任」の免責文言
├ 不良品の場合は「景品交換」で対応（返金不可）
└ 交換手続きを明確に表示

【チェック】
□ 店内掲示板・ HP に交換ポリシー明記
□ スタッフの説明スクリプト標準化
```

#### 5-2. 取得困難・機械故障時
```
【顧客（未成年含む）からのクレーム対応】
├ 「機械が壊れてる」 → 返金対応フロー
├ 「取れなかった」 → 代替機の提案
└ 「詐欺だ」（SNS投稿）→ 評価対応

【必要な仕組み】
□ クレーム記録用ノート（日報に記入）
□ クレーム対応フロー（店長・副店長向け）
□ Google Map・SNS監視と返信体制
```

#### 5-3. 未成年者への対応
```
【重要】
クレーンゲーム利用者には未成年が含まれる
→ 「お金の価値教育」が重要

【推奨施策】
├ 「お小遣いの範囲内で遊びましょう」の掲示
├ 親子で来店時の「家計管理の説明」
└ 依存症対策（相談窓口の掲示）

【法的根拠】
└ 特に法律で禁止されてはいないが
  「社会的責任」として対応推奨
```

### Risk 6: 個人情報保護（GDPR 類似） 🟡 [MEDIUM PRIORITY]

#### 6-1. 顧客データベースの取扱い
```
【取得するPII（個人識別情報）】
├ LINE公式アカウント登録 → 電話番号・年齢
├ 会員カード → 氏名・住所・電話番号
└ カード決済 → カード番号（PCI-DSS準拠必須）

【リスク】
├ 個人情報の漏洩
├ マーケティング利用の同意漏れ
└ 退会時のデータ削除忘れ

【CLO対応】
□ 個人情報保護ポリシーの策定
□ システムベンダーの「個人情報管理体制」確認
□ スタッフの秘密保持契約
```

#### 6-2. LINE・SNS マーケティング
```
【特定電子メール法の遵守】
ID-POS（来店際のID）取得 → 購買分析 OK
しかし、DM・セールス連絡は「事前同意」必須

【確認項目】
□ LINE公式登録時に「セール通知」の同意欄
□ 「配信停止」ボタンの用意
□ 月2回以上の配信は事前予告
```

### Risk 7: 税務申告・会計 🟡 [MEDIUM PRIORITY]

#### 7-1. 法人設立形態
```
【現在の仮定】
「回遊館グループが買収」→ 子会社 or 支店？

【選択肢】
├ A) 新設法人（GiGO我孫子店㈱など）
│     └ メリット: コーポレートガバナンス明確
│     └ デメリット: 申告手続き増加
│
└ B) 既存法人の支店化
    └ メリット: 申告簡素化
    └ デメリット: 支店責任が親会社に及ぶ
```

#### 7-2. 必須の国家申告
```
【設立時】
□ 法務局: 法人設立登記
□ 税務署: 法人設立届出書
□ 都道府県庁: 法人設立届
□ 市区町村: 事業開始届

【毎年】
□ 法人所得税申告書
□ 消費税申告書（売上>1,000万円の場合）
□ 地方法人税・事業税申告書
□ 年末調整・給与支払い報告書（給与支払い時）
```

#### 7-3. 会計書類
```
【毎月必須】
├ 日次売上レポート（機器別・時間帯別）
├ 経費の領収書保管
├ 給与計算・社会保険料納付
└ 景品原価の記録

【決算時（年1回）】
├ 決算書の作成（BS・PL）
├ 固定資産台帳の更新（460台の償却記録）
├ 税務署への申告
└ 監査（上場企業でなくても推奨）
```
