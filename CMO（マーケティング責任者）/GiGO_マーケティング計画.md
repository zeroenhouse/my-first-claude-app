# CMO - GiGO我孫子 マーケティング計画書

## 📊 CMOの責務（このプロジェクトにおいて）

### 主要業務
1. **市場分析・競合分析**の詳細化
2. **ターゲット顧客セグメント**の定義
3. **差別化ポイント** の明確化
4. **マーケティング施策** の企画
5. **カスタマージャーニー** の設計

## 🎯 優先度別タスク（Phase 1）

### ✅ 高優先度（4月中に完了）

#### Task 1: 競合分析の深掘り
**目的**: 20分圏内の競合10店舗をベンチマーク化  
**データソース**: `データ/プライズゲーム店舗リスト_392件統合完全版.csv`

**実施内容**:
- [ ] 20分圏内10店舗の詳細スペック抽出
  - 店舗名、評価、場所、業態
  - 推定売上、台数、機種構成（可能な限り）
  - スタッフ評判・口コミ分析
  
- [ ] 最大脅威「ゲームアルファ（柏マルイ）」の研究
  - 評価3.9（我孫子の3.0より高い理由）
  - フィギュア特化の戦略
  
- [ ] GiGO系列4店舗（全て2.4〜3.0）の分析
  - なぜ評価が低いのか
  - 改善可能なポイント

**アウトプット**: `CMO/競合分析_詳細版.md`

---

#### Task 2: ターゲット顧客セグメント定義
**目的**: Who を徹底的に理解する

**実施内容**:
- [ ] カスタマーペルソナ作成（3〜4ペルソナ）
  - 名前、年齢、性別、職業
  - クレーンゲーム利用頻度
  - 好む景品ジャンル（食品vs.フィギュア vs.ぬいぐるみ）
  - GiGO現状への不満点
  
- [ ] 年代別・性別の来店予測
  - 20代女性: ぬいぐるみ・食品志向
  - 20代男性: フィギュア志向？
  - ファミリー層: 日用品・食品
  
- [ ] 「GiGO低評価顧客の取込み」の具体化
  - 現在GiGO柏駅前を使ってる顧客
  - 何に不満か？（「取れない」など）

**アウトプット**: `CMO/ターゲット顧客ペルソナ.md`

---

#### Task 3: 差別化ポイント明確化
**目的**: なぜGiGO我孫子を選ぶのか

**実施内容**:
- [ ] OTAICHI岐阜大垣モデルの強み分析
  - 「消費財型」戦略（フィギュア特化と異なる）
  - 食品44% + 飲料17% + 日用品13% = 74%
  - このMix がなぜ3億円達成できるのか
  
- [ ] 立地の優位性を言語化
  - 国道6号沿い
  - スマイルフィールド（VZONEパチンコ）との相互送客
  - 駅からの距離・アクセス
  
- [ ] 「評価3.0→4.0へ」のシナリオ
  - 何をやれば評価が上がるか
  - スタッフ対応、景品品質、機器の取りやすさ

**アウトプット**: `CMO/差別化ポイント_戦略書.md`
