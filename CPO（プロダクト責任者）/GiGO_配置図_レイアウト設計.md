## 🎮 機器配置戦略

### 基本パラメータ
```
通路幅:             3,000mm
機器寸法:           幅650mm × 奥行850mm
機器間隔:           80mm
背中合わせ2列構成:  幅 [850 + 80 + 850] = 1,780mm + 通路3,000mm = 4,780mm

計算式: (22,200mm - 各側バックヤード) ÷ 4,780mm = 利用可能ユニット数
```

## 💡 配置による差別化ポイント

### 「取れやすさ」を実現する配置

1. **見やすさ**: 機器むき出しでなく、適度な間隔でストレス低減
2. **アクセス**: 通路3m確保で、どの機器にもスムーズにアクセス
3. **監視**: スタッフが全エリアを見守りやすい配置
4. **景品の見栄え**: 天井高・照明でプライズジャンル別に差別化

### その他の設計工夫
- **VZONEとの連係**: パチンコ共有トイレ利用者を歓迎
- **改装イメージ**: 「新しい」「きれい」を強調（評価向上へ）
- **混雑対応**: 2F への循環線で1F の密集緩和
