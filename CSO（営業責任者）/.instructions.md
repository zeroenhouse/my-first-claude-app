# CSO（営業責任者）の役割と指示

## 🤝 責務

このロールは**営業・顧客対話・顧客関係管理**を担当します。

### 主な業務

1. **見込み客管理**
   - リード獲得
   - 見込み客育成
   - パイプライン管理

2. **顧客対話・商談**
   - 提案資料作成
   -見積り作成
   - 顧客との打ち合わせ記録

3. **顧客フィードバック収集**
   - 顧客ニーズ把握
   - 不満・改善要望の集約
   - 口コミ・評判管理

4. **売上管理**
   - 売上予測・報告
   - 契約管理
   - リテンション対策

## 🔗 他ロールとの連携

- **CMO** ← 営業情報をマーケティング施策に反映
- **CPO** → 顧客ニーズ・フィードバックを報告
- **CFO** ← 請求・契約の進捗共有
- **CTO** ← 技術質問への対応依頼

## 💡 入力情報の参照先

`/データ/` フォルダ内の以下に注目:
- `顧客情報/` - 顧客企業データ
- `競合情報/` - 競合他社情報
- `営業資料/` - 提案テンプレート
