# CSO - GiGO我孫子 顧客戦略・営業計画

## 🎯 CSO の役割

クレーンゲーム事業の成功は、**正しい顧客をつかみ、繰り返し来店させること**に尽きる。  
本計画では新規顧客獲得と既存GiGO顧客の再来店化に注力する。

---

## 👥 ターゲット顧客セグメント

### セグメント1: 「初心者ファミリー層」
- **属性**: 20代後半〜40代、子連れ家族
- **来店頻度**: 月1〜2回（塾・買い物ついで）
- **好む景品**: 食品・飲料・ぬいぐるみ
- **課題**: 「取れない」で諦めてしまう
- **対策**: 輪っか配置・わかりやすい説明・取れやすい機種を1F に

### セグメント2: 「テクニカル玄人層」
- **属性**: 20代〜30代、男性主体
- **来店頻度**: 月2〜4回（週末の楽しみ）
- **好む景品**: フィギュア・玩具・限定グッズ
- **課題**: 「今どき の機種がない」
- **対策**: 3本爪・3本ミニの充実・新機種導入・イベント企画

### セグメント3: 「既存GiGO低評価顧客」（取込み対象）
- **属性**: 過去GiGO柏駅前・ラポート柏を利用
- **課題**: 「スタッフ対応悪い」「取れない」「古い機種」
- **対策**: 
  - スタッフ教育 & 親切対応を徹底
  - 新しい機種・景品でリピート誘導
  - リターゲティング キャンペーン（DM・SNS）

### セグメント4: 「新規層」（20代女性・学生）
- **属性**: 学生・OL・カップル
- **来店頻度**: 月1回程度（デート・気分転換）
- **好む景品**: ぬいぐるみ・玩具・食品
- **対策**: SNS＆インスタ映え演出 & ビギナーフレンドリー配置

## 💡 顧客獲得戦略

### Phase A: プレオープン（オープン前6週間）

#### Task A1: 既存GiGO顧客への直接アプローチ
- **対象**: GiGO柏駅前・ラポート柏の過去来店者データ
- **施策**:
  - DM送付（「新しくなります」案内）
  - LINE公式アカウント登録キャンペーン
  - プレオープン体験招待（限定20人×3日）
  
- **期待効果**: 既存顧客200人×月2回来店 = 月400人

#### Task A2: SNS・タウンメディア戦略
- **施策**:
  - Instagram開設 → 「新機種」「景品紹介」毎日投稿
  - Facebook → 地元コミュニティに告知
  - タウンメディア（地元掲示板・フリペ）に記事化
  - YouTube → オープニング動画（機種紹介・景品ショーケース）
  
- **期待効果**: 新規認知層 月500人

#### Task A3: 競合店への「顧客請奪」キャンペーン
- **対象**: ゲームアルファ・タイトーステーション等の来店者
- **施策**:
  - チラシ配布（競合店周辺・駅）
  - Google Local Guides への口コミ戦略
  - 「評価＋1を目指す」利用者体験設計
  
- **期待効果**: 新規顧客 月800人

---

### Phase B: オープン初期（オープン後1ヶ月）

#### Task B1: リピート化ドライブ
- **LoyaltyDM**: 来店者に「次来店時 -100円クーポン」配布
- **SNS連携**: Instagram フォロー者に毎週キャンペーン
- **イベント**: 週末イベント（「今週の景品ピックアップ」など）

#### Task B2: Google評価向上キャンペーン
- **来店者に依頼**: 「評価・コメント投稿してくれた方にコイン50枚進呈」
- **目標**: 3.0 → 3.8超を3ヶ月で達成
- **継続施策**: 悪い評判への丁寧な返信・改善

---

### Phase C: 安定期（3ヶ月以降）

#### Task C1: 固定客化
- **会員制度**: LINE登録会員に「ポイント」付与
- **固定グループ構築**: 「毎週金曜は我孫子」という習慣化
- **イベント化**: 月1回の「新機種 & 新景品」発表会

#### Task C2: 発展促進
- **ファン育成**: 特ファン（月5回以上来店者）に「VIP待遇」
- **紹介制度**: 「友人紹介で両者に -500円」キャンペーン
- **季節イベント**: GW・お盆・年末に大型キャンペーン

## 🎤 顧客接点・対応マニュアル

### スタッフ教育の最重要ポイント

#### 1. 「取れにくいお客様」への対応
```
❌ NG: 「その機種は難しいですよ」
✅ OK: 「このコツを試してみてください」

→ スタッフが実演で2〜3回取ってみせる
→ 「今のやり方をもう一度」と励ます
→ 取れたら「取れました！」と客と一緒に喜ぶ
```

#### 2. 景品の説明
```
機種選択時に「この機種、今こんな景品です」を見やすく説明
→ 的確な機種選択 → 取れる実感 → リピート
```

#### 3. クレーム対応
```
例: 「機械が動かない」
✅ 手順:
  1. 謝る（「申し訳ないです」）
  2. すぐ直す（or 代替機を用意）
  3. お詫び景品（100円分コイン）
  4. 最後に「ありがとうご利用ください」
```
